# Spring Boot + React E-Commerce Application

A full-stack E-Commerce application developed using Spring Boot and React that allows users to manage products with complete CRUD functionality, image uploads, and PostgreSQL database integration.

---

## Features

- Add Products
- Update Products
- Delete Products
- View All Products
- View Product Details
- Upload Product Images
- Display Images from Database
- REST APIs
- PostgreSQL Integration
- Responsive React UI

## API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /api/products | Get all products |
| GET | /api/product/{id} | Get product by ID |
| GET | /api/product/{id}/image | Fetch product image |
| POST | /api/product | Add new product |
| PUT | /api/product/{id} | Update product |
| DELETE | /api/product/{id} | Delete product |
